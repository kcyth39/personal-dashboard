# Dashboard Development Diary

こんにちは。この記録は、AIエージェントと一緒に非エンジニアが自分専用の「パーソナルダッシュボード」を作り上げていく過程をまとめた開発日記です。

## Day 1: プロジェクトの立ち上げ

今日はプロジェクトの第一歩として、開発環境の準備と基本レイアウトの作成を行います。

**目標機能**:
- ReactとTailwind CSSを使ったモダンな画面作り
- ニュース、カレンダー、リンク集のカードが並ぶシンプルなデザイン

AIエージェントに「ReactとTailwind CSSを使って、ローカルで動作するパーソナルダッシュボードを作って」と指示を出しました。これにより、一から開発環境が構築され、コンポーネントの骨組みが作られていきます。

### トラブル発生：Node.jsが見つからない！？
開発をスタートして最初のステップであるプロジェクトの作成をAIが試みたところ、コマンドが見つからないというエラーが発生しました。
AIが自律的にMacの中にある複数のディレクトリ（`/usr/local` や `/opt/homebrew` などの開発ツールが置かれやすい場所）を捜索した結果、そもそも開発に必要な「Node.js」自体がPCにインストールされていないことが判明しました。

非エンジニアのPC特有の「前提となる開発ツールが入っていない」という壁に初日からぶつかりました。AIから「まずはNode.jsをインストールしてください」と具体的な案内をもらい、環境構築の第一歩を踏み出すことになりました。

### そして完成した「自分だけのための朝礼画面」
気を取り直して再開すると、AIはあっという間にReactプロジェクトを生成。「Tailwind CSS」というモダンなデザインツールを駆使し、ガラス細工のように透き通った『グラスモーフィズム』なカードデザインを作ってくれました。

現在できあがっているのは：
- ニュースの枠（将来的にはAPIで取得予定）
- 今日が何日か一目でわかるカレンダー
- 毎日の仕事やポートフォリオに一発で飛べるクイックリンク集
- 毎日変わる（？）TIPSカード

これらが美しいグラデーションの背景の上に並んでいます。

### 「朝起きて一番に見る」ための工夫
「毎回コマンドを打って立ち上げるのは面倒。ブラウザを開いたらすぐに見たい」というワガママなお願いに対しても、AIは完璧に答えてくれました。
「PM2」というプロセスマネージャーを導入し、Macの背後で常にこのダッシュボードが動き続けるように設定。あとはブラウザのホームページ設定を `http://localhost:5173` にするだけで、「起きて・ブラウザを開くだけ」の完璧な朝のルーティンが完成しました。

## Day 1.5: トラブルシューティング - フォルダ移動と「プロセス迷子」

ダッシュボードを本番運用するため、一時作業用のフォルダから正式な作業フォルダ（`Antigravity/Personal-dashboard`）にプロジェクトをドサッと移動させました。すると、ここで新たな問題が発生！

**問題発覚**
「フォルダを移動しました。一緒に発生した問題をデバッグしてください」とAIに伝えたところ、プロ目線の調査が始まりました。画面自体は見えているのに、なぜか動作がおかしい。

**AIのアクションと解決のやり取り**
1. **ブラウザでの目視確認とログ調査:** AIはまず仮想ブラウザ機能を使って `localhost:5173` にアクセスし、エラーの有無を確認しました。
2. **真犯人の特定 (`pm2` の状態確認):** AIがターミナルで `npx pm2 status` コマンドなどを叩いて確認した結果、バックグラウンドで動かしていた「PM2」というプロセスが、前の古いフォルダを参照したまま動いている（プロセス迷子になっている）ことが判明しました。
3. **プロセスの削除と再起動:** 古いプロセスを `npx pm2 delete personal-dashboard` で削除し、新しい場所で再度 `npx pm2 start npm --name "personal-dashboard" -- run dev` を実行。これにより新しいフォルダでの起動に成功しました。
4. **ついでにお掃除 (`npm run lint`):** さらにAIは気を利かせてコードの自動解析を行い、カレンダーコンポーネントに残っていた「使われていない不要な変数」を削除して、コードを綺麗にしてくれました。

**まとめ**
非エンジニアだと「フォルダを移動させただけなのに、なぜかうまく動かない…」と途方に暮れてしまいますが、PCの裏側では古いパスをプログラムが握りしめたまま走り続けていることがある、という裏側を知るいい機会になりました。「移動したから見ておいて！」と放り投げるだけで、原因特定から再起動、さらにお掃除までパパッと終わらせてくれるAIは本当に心強い相棒です。

## Day 2: 動的な情報の息吹 - ニュースとマーケットの連携

ついにダッシュボードに「生きた情報」が流れ込み始めました！これまでは単なるデザインの枠組みでしたが、今日からは実際のニュースや株価、為替レートが表示されます。

**今回実装したこと**
- **4大市場指標の表示:** 日経平均、S&P 500、金(XAU)、ドル円をリアルタイムに近い形で表示するようにしました。
- **Googleニュース連携:** 常に最新のトップニュースが10件並ぶようにしました。
- **賢い更新システム:** 「ニュースは30分ごと、マーケットは5分ごと」という要望に合わせて、AI相棒が裏側で動く取得スクリプトを組んでくれました。取得時間も表示されるので鮮度も一目瞭然です。

**AIとの連携で感動したポイント**
「ブラウザから直接ニュースを取ろうとすると、セキュリティ（CORS）の関係でエラーになる」という技術的な壁がありました。AIはすぐに「それならMacの裏側（Node.js）でデータを取ってファイルとして書き出しましょう」と代替案を出し、さらにはPM2という常駐プログラムの設定まで手際よくこなしてくれました。

今や、ブラウザを開くだけで「今、世界で何が起きているか」「市場はどう動いているか」が秒でわかります。「単なるメモ帳」から「自分専用の司令塔」へと進化した瞬間でした。

さらに、カレンダーの上に「天気」も追加しました！
Open-Meteoという、APIキー不要でサクッと使えるサービスをAIが提案してくれました。
緯度と経度を設定ファイルに書くだけで、その場所の空模様、今の気温、最高・最低気温、さらには湿度まで完璧に表示されます。
これで朝、家を出る前に傘が必要かどうかもこのダッシュボード一枚で完結です。
さらに、このダッシュボードは「生きて」います。
場所を東京から大阪に変更しただけでなく、時間帯に合わせて挨拶や背景がリアルタイムに変わるようになりました。
朝から夕方までは目に優しい「日本の自然」が広がり、日が暮れるとAIが描いた「活気あふれる大阪の夜景」へと自動的に切り替わります。真夜中には、静寂を感じさせる深い夜のイメージが心を落ち着かせてくれます。

挨拶も「Good Morning」一辺倒ではなくなりました。
「今日も世界を驚かせる準備はできていますか？」「夜の静かな時間は、深い思考に最適です」
まるで気の利いた秘書が隣にいるような、そんな温かみのあるダッシュボードに仕上がりました。
技術的な「ツール」を超えて、自分の生活リズムに寄り添ってくれる「相棒」ができたような気分です。
